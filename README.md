# portal-ued-notes

本仓库用于归档和发布多个独立的 HTML 页面，主要服务于前端用户体验相关的笔记与演示。仓库本身不绑定任何特定域名或网站，所有页面均为自包含的静态资源。

## 项目简介

portal-ued-notes 是前端 UED（用户体验设计）领域的技术笔记和页面示例集合。每个 HTML 页面独立存在，可单独打开查看，适合快速演示、原型验证或技术备忘。

仓库内容可能包括但不限于：
- 交互样式演示
- 组件用法示例
- 前端技术实验
- 设计思路记录

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 共用资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个 HTML 文件均可独立运行，相互之间无依赖。
- `assets/` 目录存放页面公用的静态资源，部分页面也可能自带内联资源。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开，无需后端服务。
- 页面内容可能包含实验性代码或临时记录，不保证长期稳定或兼容所有浏览器。
- 仓库会不定期更新，新增或修改页面，历史版本可通过 Git 提交记录追溯。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面内容或修复问题。
- 请确保新增页面为独立 HTML，并尽量保持目录结构清晰。
- 不建议在页面中加入外部跟踪、广告或非必要的网络请求。
- 仓库维护者会定期检查并清理过时或无效的页面。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接用浏览器打开 `pages/` 下的任一 HTML 文件。
3. 或通过 GitHub Pages 等静态托管服务发布整个仓库，即可在线访问。
